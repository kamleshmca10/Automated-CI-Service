cassandra
====================

# Build
To build a new cassandra container

```sh
%> docker build --rm -t cassandra .
```

# Run
To run the new cassandra container

```sh
%> docker run cassandra
```

# Access
This docker container exposes the following TCP/UDP ports

TCP
* 22
* 7000
* 7199
* 9042
* 9160
* 61621
